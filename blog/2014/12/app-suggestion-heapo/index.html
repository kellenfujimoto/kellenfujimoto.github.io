[App Suggestion] Heapo

Mon, Dec 22, 2014 ~100 words

Cross-platform clipboard management is a tough problem to solve. Mohammad Khoman’s Heapo has the right features — a today view widget, sync between iOS, Windows, and Mac, ad-free interface — all in a free app. As users of Tapbot’s venerable Pastebot know, being able to quickly save clipboard contents for later1 is incredibly useful: especially when it’s a swipe away.

I highly recommend giving Heapo a try.
